# 表达式

关键字运算符

| 关键字     | 含义                       | 说明                     |
| ---------- | -------------------------- | ------------------------ |
| typeof     | 取变量或值的类型           |                          |
| void       | 运算表达式，忽略值         |                          |
| new        | 创建指定类的对象实例       |                          |
| in         | 检查对象及其原型链的属性   |                          |
| instanceof | 检查对象是否为指定类的实例 | 指定类可以是扩展链中的类 |
| delete     | 删除实例属性               |                          |
| yield      | 从生成器内部返回一个值     |                          |
| await      | 在异步函数内等待一个值     |                          |

值运算符

| 按结果值的类型 | 运算符                           | 目标类型 | 说明                   |
| -------------- | -------------------------------- | -------- | ---------------------- |
| 数值运算       | [+,-,\*,/,%,++,--]               | Number   | 一般表达式运算         |
| 位运算         | [~,&,\|,^,<<,>>,>>>]             | Number   | 一般表达式运算         |
| 逻辑运算       | [!, \|\|, &&, ?:, (优先级), ","] | Boolean  |                        |
| 字符运算       | [+]                              | String   |                        |
| 函数调用       | [identifier()]                   | Function |                        |
| 对象           | [.,[],new,in]                    | Object   |                        |
| 等值检测       | [==,===]                         | -        |                        |
| 赋值运算       | [=,+=,-=,*=,/=,%=,<<=,>>=,>>>=]  | -        | 仅是简写，不会提升性能 |

运算符的优先级：建议使用()优先级符进行分组，不必记忆
