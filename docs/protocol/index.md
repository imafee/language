# Protocal

network

- HTTP(1/2/3)
- SSE 协议:Server-Sent Events,服务器发送事件。用于实现服务器主动向客户端推送数据。
- SSL/TLS
- QUIC (Quick UDP Internet Connections),UDP basd 的传输层网络协议。旨在减少连接建立的延迟,提高传输速度。HTTP/3 基于 QUIC 协议。
- WebSocket
- WSS WebSocket Secure,是 WS(WebSocket)协议的安全版本,类似于 HTTPS,它在 TCP 之上添加 SSL/TLS 层,用于加密 WebSocket 连接和传输的数据
- TCP/IP
- UDP
- ICMP
- DNS
- SSH
- SMTP/POP3/IMAP
- MIME
- RTSP 协议:Real Time Streaming Protocol,实时流媒体传输协议
- RTC 协议:Real-time Communications,真实时间通信
- XMLHttpRequest 用于在后台与服务器交换数据,是 AJAX 的基础
- CORS 跨域资源共享协议,允许网页从其他域请求资源。需要浏览器和服务器同时支持
- JSONP 通过`<script>`标签加载跨域 JSON 数据。JSONP 依靠 JavaScript 同源策略的漏洞实现跨域数据请求
- WebRTC 支持浏览器之间的点对点通信,可用于 Web 视音频通话等
- gRPC 协议:是一个高性能、开源的 RPC 框架,面向移动和 HTTP/2 设计。gRPC 基于 HTTP/2 标准设计,使用 Protocol Buffers 作为接口描述语言。
