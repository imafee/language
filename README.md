## operating locally

edit document or run demos locally

- clone this repository
- install dependencies

```shell
# Nodejs needs to be installed
proj> npm install
```

- npm scripts

```shell
npm run docs:dev # edit document
npm run stdlib # run demos on autumatic opening webBrowser
```
